# Gazella Media Service #

## Ejecutando el servicio ##

### Prerequisitos ###

* Necesita tener configurado y corriendo el [API Gateway de Gazella](https://github.com/AbelYong/gazella-api-gateway).
* Necesita tener instalado **Docker** y **Docker Compose** en su entorno para levantar el servidor web, el almacenamiento de MinIO y el broker de RabbitMQ.

### Pasos ###

Clone el repositorio y ejecute (opcional para tener auto-completado y desarrollo local):
